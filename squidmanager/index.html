SquidManager is a Bank Editor for the ALM/Busy Circuits Squid Salmple, an 8 channel sampler for Eurorack.

  • Open source and cross platform (Windows and OSX)
  • Edit Banks from USB Drive
  • Drag and drop samples
  • Drag and drop multiple samples on Cue Sets to concatanate and set a Cue Set for each
  • Audition raw samples to adjust loop points
  • Clone Parameters from one to channel to another
  • Loop end points align view
  • Full waveform view with loop point edit markers
